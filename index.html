Correlation vs Causation

1. Correlation is used to show the relationship between two variables, but it must be linear. Non linear functions cannot have correlation because it is a curve.

2. Causation is based off of looking at the explanatory variables as the causes for change on the response variables. Correlation does not imply causation but can be used as clues if solid enough. The only way to determine causation is to run randomized controlled experiments that can rule out extraneous factors.
